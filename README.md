### To store and use configuration files across multiple machines.

Configuration files for: 

 * Vim,
 * Zsh (_Z Shell_),
 * tmux,
 * Secure Shell _(SSH)_  Multiple SSH Identities.


### Usage

`cd ~ `

`ln -s ~/configs/.vim . `
`ln -s ~/configs/.vimrc . `
`ln -s ~/configs/.zshrc . `
`ln -s ~/configs/.tmux.conf . `

`cd ~/.ssh `

`ln -s ~/configs/.ssh/config . `
